# 内容审核与合规规则

> 本文档覆盖 Sloth-SMM-Eido 运营过程中涉及的内容审核规则，包括广告法、版权、平台原创声明及各平台特殊规范。

---

## 1. 通用合规原则

### 1.1 广告法（中国）

**绝对化用语禁用清单**：
- 国家级、最高级、最佳、第一、唯一、首创、领先、顶级、极品、终极、独家、万能、永久、百分百、百分之百
- 替代方案：使用"非常""很""相当""比较"等相对化表述，或提供具体数据支撑

**虚假宣传风险点**：
- 不得虚构数据、案例、用户评价
- 不得使用无法验证的统计数据（如"90% 的人不知道"需有来源）
- 不得对商品/服务的性能、功能、质量作虚假或引人误解的宣传
- 涉及疗效、保健功能的，必须有明确资质证明

**特殊行业附加限制**：
- 金融：不得承诺收益率，不得使用"保本""稳赚"等表述
- 医疗：不得涉及疾病治疗功能宣传
- 教育：不得对升学、考试效果作出明示或暗示的保证性承诺

### 1.2 版权与知识产权

**图片版权**：
- AI 生成配图：需确认生成平台的使用许可，保留生成记录
- 网络图片：优先使用 CC0 或明确可商用的图库，禁止直接抓取他人作品
- 截图/引用：涉及第三方界面截图时，需做脱敏处理，去除敏感信息

**文字版权**：
- 引用他人观点、数据必须标注来源
- 翻译内容需注明原文出处
- 避免大段复制他人文章内容（连续 200 字以上雷同可能触发平台原创检测）

**字体版权**：
- 公众号/头条等平台默认使用系统字体，通常无版权风险
- 如需使用特殊字体，确认已获得商用授权

### 1.3 个人信息保护（PIPL）

- 不得公开披露可识别个人的信息（真实姓名、电话、地址、头像等）
- 客户案例中的个人信息必须脱敏处理
- 引用用户评论/反馈时，使用化名并去除可识别信息

## 2. 平台-specific 规则

### 2.1 微信公众号

**原创声明**：
- 发布到公众号的内容建议声明原创（获得原创保护）
- 转载他人文章需获得授权，并标注来源
- 引用比例不得超过原文 30%，否则可能无法通过原创审核

**敏感内容**：
- 禁止涉及政治、色情、暴力、赌博等内容
- 金融类内容需额外注意合规，避免被判定为"诱导分享/关注"
- 医疗、保健品内容需谨慎，避免被平台限流

**广告标识**：
- 商业推广内容需标注"广告"（如开通流量主后的底部广告自动标注）
- 软文推广建议在文末声明利益相关

### 2.2 今日头条 / 百家号

**算法友好规则**：
- 标题避免纯标题党（高点击低完读会被算法降权）
- 内容必须原创，抄袭/洗稿会被限流或封号
- 图片需清晰、无水印、无二维码（二维码会被平台屏蔽或替换）

**审核严格度**：
- 头条对政治、军事、社会热点类内容审核较严
- 健康、财经类内容可能需要资质认证

### 2.3 知乎

**社区规范**：
- 禁止"挂人"、人身攻击、引战
- 商业推广需标注"利益相关"或"合作推广"
- 避免"一句话回答"或纯引流内容（会被折叠或删除）

**原创要求**：
- 知乎对抄袭检测较严格，引用需明确标注
- 回答被标记为"转载"会影响账号权重

### 2.4 小红书

**社区公约**：
- 禁止虚假种草、夸大功效
- 医疗美容类内容需要平台认证
- 避免直接引流（如放微信号、二维码），可用谐音或引导到主页

**审核特点**：
- 机器审核为主，关键词触发人工复核
- 常见限流词：最、第一、绝对、根治、治愈、特效、纯天然、零添加等

## 3. 合规检查清单

每篇文章发布前，执行以下检查：

- [ ] 无绝对化用语（广告法）
- [ ] 数据/案例有来源或标注为虚构（反虚假宣传）
- [ ] 个人信息已脱敏（PIPL）
- [ ] 引用内容已标注出处（版权）
- [ ] 商业推广已声明利益相关（平台规则）
- [ ] 图片无侵权风险（版权）
- [ ] 无平台敏感词（各平台-specific）

---

## 4. 违规处置流程

1. **发布前检测**：技能内置合规引擎自动扫描，标红违规项
2. **发布后监控**：关注平台通知，如收到"内容违规"通知，立即下架或修改
3. **申诉渠道**：如认为误判，可通过平台官方申诉渠道提交申诉
4. **记录归档**：将违规案例记录到 `00_Shared/05_Best_Practices/compliance_cases.md`，避免重复犯错
