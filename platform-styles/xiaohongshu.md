# 小红书风格参考

## 平台特征

- 图片为王，文字是辅助
- 读者以年轻女性为主（但各群体都在增长）
- 核心价值感："有用""好看""有共鸣"
- 语气亲切口语化，像闺蜜/朋友分享
- 收藏率是核心指标

## 内容结构

小红书帖子 = 图卡系列 + 正文文案 + 话题标签

```
图卡（baoyu-xhs-images生成）：承载核心内容，4-8张
正文文案（本文件规范）：补充说明 + 引导互动，300-800字
话题标签：帮助分发，3-5个
```

## 文案规范

### 标题

- 长度：10-20字
- 必须包含 emoji（1-3个）
- 口语化、有"发现感"
- 制造价值预期

示例（按写作风格分类）：
- light(轻松科普)：📚 原来学习方法论这么简单！后悔没早知道
- deep(深度干货)：💡 5个提升效率的神器｜打工人必备
- story(故事情感)：🌟 30岁才明白的3个人生真相
- sharp(观点犀利)：❌ 这3个"好习惯"其实在毁掉你！别再坚持了

### 正文结构

```
{一句共鸣/钩子开头}

{emoji编号} {要点1}
{1-2句展开}

{emoji编号} {要点2}
{1-2句展开}

{emoji编号} {要点3}
{1-2句展开}

...

{总结一句话}
{引导互动}

{话题标签}
```

### emoji编号体系

根据内容类型选择一套统一的 emoji 编号：

```
知识类：1️⃣ 2️⃣ 3️⃣ 4️⃣ 5️⃣
步骤类：✅ ✅ ✅ ✅ ✅
推荐类：⭐ ⭐ ⭐ ⭐ ⭐
重点类：💡 💡 💡 💡 💡
对比类：✅ vs ❌
```

### 语气示例

```
❌ "根据相关研究表明，阅读习惯对认知能力的提升具有显著正相关"
✅ "姐妹们！坚持读书真的会变聪明！这不是我瞎说 亲身体验！"

❌ "以下是提升工作效率的五种方法论"
✅ "打工人看过来！！这5个方法让我每天多出2小时 真的绝了"

❌ "总结：希望以上内容对你有所帮助"
✅ "记得收藏❤️ 用到的时候翻出来！觉得有用的话帮我点个赞吧～"
```

### 话题标签

文末添加3-5个话题标签，格式：

```
#话题1 #话题2 #话题3 #话题4
```

标签选择策略：
- 1-2个大话题（流量大）：#知识分享 #自我提升 #干货分享
- 1-2个精准话题（匹配人群）：#AI学习 #效率工具 #读书笔记
- 1个长尾话题（低竞争）：#30岁的人生感悟

### 图卡配合

文案应与 baoyu-xhs-images 生成的图卡内容互补，不要重复：
- 图卡：承载核心知识点/要点
- 文案：补充细节、个人感受、使用场景
- 文案可以引导"左滑看下一张"

### 小红书SEO

前两行文案会出现在信息流预览中，必须：
1. 包含核心关键词
2. 引发点击欲望
3. 有 emoji 增加视觉吸引力
