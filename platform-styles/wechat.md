# 微信公众号风格参考

## 排版规范

### 标题

- 长度：15-64字
- 风格：制造好奇心缺口、善用数字、善用「」强调
- 可用副标题补充信息

示例：
- 深度型：「万字长文」从GPT到Agent：AI应用落地的3个关键拐点
- 故事型：35岁那年，我放弃了年薪百万的工作
- 犀利型：别再鼓吹"自律改变人生"了，这根本不是真相
- 科普型：原来手机里这个功能，90%的人都没用过

### 正文格式

```
段落长度：2-3句/段
段落间距：视觉上要有呼吸感
小标题频率：每500-800字一个
小标题风格：可以用 emoji/符号装饰，但不要过度

金句处理：
- 可单独成段
- 可加粗
- 不超过20字为佳

引用处理：
- 外部链接 → 底部引用编号（baoyu-post-to-wechat 自动处理）
- 重要引用 → 用引用块格式

图片处理：
- 图片居中
- 图片下方加一行灰色小字说明
- 首图建议：与标题主题相关的高质量图
```

### 文末引导

```markdown
---

如果觉得这篇文章对你有启发，
欢迎**点赞、在看、转发**，让更多人看到。

关注我，持续输出{领域}的深度思考。
```

### 微信主题配置

使用 baoyu-post-to-wechat 的主题系统：

| 主题 | 风格特点 | 适合文章类型 |
|------|---------|------------|
| grace | 优雅、大方、适合深度内容 | deep, story |
| simple | 简洁、干净、聚焦内容本身 | sharp, light |
| modern | 现代感、科技感 | deep(科技类) |
| default | 通用样式 | 通用 |

颜色预设：blue(默认), green, orange, red, purple, brown, pink, grey, darkblue, gold, olive, cyan, darkgreen
