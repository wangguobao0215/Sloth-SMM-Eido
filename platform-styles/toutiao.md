# 今日头条风格参考

## 平台特征

- 算法推荐为主，标题和第一段决定推荐量
- 读者跨年龄层、偏大众化
- 完读率是核心指标（高点击+低完读 = 被限流）
- 图片权重高，首图决定封面

## 排版规范

### 标题

- 长度：15-30字
- 核心要求：包含文章核心信息，不能纯悬念
- 关键词前置：核心关键词放在标题前半段（帮助算法分类）
- 避免：纯标题党、全是感叹号、夸张失实

示例：
- 深度型：AI Agent实战：从概念到落地的3个核心问题和解法
- 故事型：从月薪5千到年入百万，这个90后做对了什么
- 犀利型：为什么我劝你不要盲目考研？这3个现实很扎心
- 科普型：手机充电的5个错误习惯，第3个很多人每天都在犯

### 第一段（极其重要）

头条的算法会根据第一段内容决定推荐人群。第一段必须：

```
1. 包含核心关键词（2-3个）
2. 概括文章核心价值
3. 引发继续阅读的欲望
4. 长度：100-150字
```

模板：
```
{核心事件/现象一句话概括}。{为什么值得关注}。{本文将提供什么}，{一句钩子}。
```

### 正文格式

```
段落长度：1-3句/段（比公众号更短更碎）
小标题频率：每400-600字
小标题风格：直白、信息性强，不要太文艺

图片规范：
- 首图必须有（决定封面）
- 每500-800字插入一张图
- 图片后添加描述文字（头条支持图片描述）
- 横版图优先（3:2或16:9）

格式技巧：
- 适当使用分隔线划分大段落
- 关键信息加粗
- 数据用阿拉伯数字（更醒目）
```

### 文末

```markdown
---

你怎么看？欢迎在评论区说说你的观点。

觉得有用就点个赞，你的支持是我创作的最大动力。
```

### 关键词布局策略

```
标题：包含1-2个核心关键词
第一段：包含2-3个关键词（自然融入）
小标题：各包含1个相关关键词
全文：核心关键词出现5-8次（不要堆砌）
```
