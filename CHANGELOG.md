# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-17

### Added

- Initial release of Sloth-SMM-Eido Community Edition (desensitized from full edition)
- Three content creation scenarios: hot topic tracking, series planning, topic-based writing
- Five-module pipeline: topic engine, writing engine, platform adapter, format engine, publish pipeline
- Multi-platform support: WeChat Official Account, Toutiao, Zhihu, Xiaohongshu, WeChat Moments
- WeChat API/browser publishing with theme support (grace, simple, modern, default)
- Toutiao CDP automation via Playwright
- Feishu (Lark) wiki archiving support
- Obsidian local vault archiving with Chinese folder naming
- Four writing styles: deep, story, sharp, light
- Configurable brand image management for WeChat articles
- AI-powered article illustration via baoyu-article-illustrator
- Xiaohongshu image card generation via baoyu-xhs-images
- Publish logging system
- EXTEND.md template with all configuration options documented
- assets/README.md guide for brand image setup

### Note

This is the Community Edition with all personal brand assets and private configurations removed. Users should customize EXTEND.md with their own settings before use.
